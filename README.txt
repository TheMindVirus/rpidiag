Thank you for downloading the Raspberry Pi Diagnostics source code package!
___________________________________________________________________________

RPiDiag.exe is a Serial Monitor written in C# .NETv4 WindowsForms [AnyCPU].
To use it you will need an RS-232 HAT or GPIO14-TXD, GPIO15-RXD and GND.

It is preconfigured to listen on COM1 at 115200 Baud on program startup.
This can be changed in the settings page for each instance of the program,
along with the window transparency option for use on white backgrounds.

We hope you enjoy using it.
